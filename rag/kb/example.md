## Product Redemption

### Gift Card Type A
To redeem: go to the platform website, navigate to redeem codes, enter your 12-digit code.
Common errors: region mismatch (check the card region matches your account region).

### Gift Card Type B
To redeem: open the platform app, go to settings, select redeem code, enter your code.
Common errors: code already used (check purchase confirmation email for original code).

## Refund Policy
Refunds are available within 7 days of purchase if:
- The code was not delivered
- The code does not work and platform support confirms it is invalid
Refunds are NOT available once a code has been successfully redeemed.

## Contact & Escalation
For urgent issues (payment disputes over $50, suspected fraud):
- Email: support@example.com
- Response time: 1 business day

For all other issues, the support agent handles them directly.
